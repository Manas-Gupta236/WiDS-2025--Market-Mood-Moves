# WiDS 2025 – Market Mood and Moves

This repository contains the code, reports, and resources developed as part of the WiDS 2025 Market Mood and Moves project.

## Project Structure
- `week1/` – Basic Python, market data handling, and sentiment analysis foundations
- `week2/` – Transformer-based sentiment analysis and exploratory analysis
- `notebooks/` – Jupyter notebooks for testing and experimentation
- `readings/` – Reference PDFs provided during the program

## Notes
- Code and reports reflect progress made during Weeks 1 and 2.
- API keys and environment variables are excluded from version control.
